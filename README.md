# Multi-Agent-Content-Production-Studio
An AI team that reads my course documents to research, write, and edit professional articles automatically. It uses Python and Google Gemini to show how multiple AI agents can work together as a digital office.

# Multi-Agent Content Production Studio

## Overview
This project is a multi-agent AI system that reads documents and automatically generates high-quality written content using Generative AI.

## Agents
- Research Agent: Extracts information from documents
- Writer Agent: Generates content using Gemini Pro
- Editor Agent: Refines and improves the content

## Technologies Used
- Python
- Google Gemini Pro API
- Google Colab
- Kaggle
- PyPDF2

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create a .env file and add:
   GOOGLE_API_KEY=your_api_key_here

## Running the Project
Run:
python main.py

## Output
The final article will be saved as:
final_article.txt
